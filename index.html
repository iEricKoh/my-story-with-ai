Tools #3 & #4

Antigravity & Windsurf

AI-Powered IDEs

Both are full AI-powered IDEs with agentic workflows, inline edits, and multi-file context. Our company provides Windsurf for free. Antigravity (by Google) is free in public preview.

Honestly, they're quite similar. But Antigravity stands out for frontend work with built-in browser debugging — agents can launch your app, interact with the UI, and verify changes automatically. It also lets the AI create its own plan before executing.

Core concept

What Are AI Skills?

Skills are specialized knowledge packs that extend AI capabilities. They teach AI how to handle specific domains — like giving it a playbook. Think of them as reusable expertise that any team member can leverage.

Another real-world skill

Jira Ticket Task Breakdown

A multi-agent pipeline that breaks down Jira tickets into concrete frontend tasks with calibrated estimates.

Multi-Agent Pipeline

✓
Gather Jira ticket via Atlassian MCP
✓
Requirements Analyst Scope table + task breakdown
✓
Scout Reusable code subagent
✓
Architect Blast radius subagent
✓
Senior Estimator Calibrates using findings
✓
OpenSpec Output Proposal, design & specs
Only 2 of 4 roles use subagents (Scout & Architect need 40-50 tool calls each). Analyst & Estimator run on main agent — saves ~70s.

Real-world skill

Automated QA Handoff

Deploy to staging, run /mcd-qa-handoff MDX-11961 — AI gathers context from Jira and git, writes testing instructions in plain language, then a reviewer sub-agent quality-checks before you paste it into Jira.

1
Gather Context Jira ticket + git log + code diff
2
Build Comment QA-friendly, non-technical language
3
Review Sub-agent catches jargon & validates
4
Present Output Copy-paste markdown for Jira
Reviewer sub-agent scans for leaked variable names, file paths, and technical jargon — QA always gets human-readable instructions.
